# CS193p - Memorize, Set, and EmojiArt

This repository contains the **Memorize**, **Set**, and **EmojiArt** examples from the Stanford [CS193p - Developing Apps for iOS](https://cs193p.sites.stanford.edu) course taught in 2021, but updated to use iOS 16 APIs and newer Swift language features.

These code examples cover all of the material from the first 12 lectures and 6 assignments (required tasks only).

> ⚠️ This code is only provided for self-learners and may deviate from what is required for Stanford students.
